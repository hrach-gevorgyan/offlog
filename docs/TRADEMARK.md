# Offlog — Trademark & Brand Notice

The MIT license in this repository covers the **source code only**. It
does not grant any right to use the **"Offlog" name, wordmark, tagline,
or icon** ("the Marks").

## What this means in practice

- **Forks must rename.** You're free to fork, modify, and redistribute
  the code under MIT — including for commercial use — but a fork may
  not keep the name "Offlog," the icon, or the tagline "Off the cloud,
  still logged." Pick your own name and mark.
- **The Marks are reserved**, established by first use as documented in
  [BRAND.md](../docs/BRAND.md) (name, tagline, and icon usage history)
  and this repo's own commit history/release tags.
- **No trademark has been formally registered.** These are common-law
  rights based on actual, dated use — the brand kit and changelog serve
  as that record.
- **No business model today doesn't waive brand rights.** Offlog is
  free and open-source with no monetization plan, but keeping the code
  license (MIT) and the brand (name/mark) separate preserves the option
  to sell, license, or hand off the project/brand later without that
  option being given away for free by implication.

## Using the name/mark

Fine without asking: referring to Offlog by name in an article, a
comparison, a "built with"/compatibility note, or an unmodified
redistribution of an official release.

Not fine without asking first: using the name or icon on a fork, a
competing product, merchandise, or anything that could look like an
official Offlog release or an endorsement by its owner.
