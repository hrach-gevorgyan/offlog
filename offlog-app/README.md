# Offlog (app source)

This is the Vite/Svelte/Capacitor project source. For full setup instructions, feature list, data model, and version history, see the [root README.md](../README.md).

Quick reference:

```bash
npm install
npm run dev              # http://localhost:5173
npm run build             # production build → dist/
npx cap sync android      # sync web build into the Android project
npx cap open android      # open in Android Studio
```

See [TECH.md](TECH.md) for architecture and internals.
